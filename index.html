    • Hvem er kurset egnet for?

    Introduksjonkurset er laget for de skal starte på programmeringsstudier og de som ønsker å se om koding er noe for dem før forplikter seg til noe.

    Hvor kan jeg lære mer?

    Hvis du har fullført kurset anbefaler jeg deg å søke studieplass på universiteter eller høyskoler. Hvis du er flink med studieteknikker og har erfaring med å studere så finnes også materiale ute på nettet, f.eks. Youtube, Udemy og Scrimba.

    Hvordan er parprogrammering?

    Når vi parprogrammerer så programmerer vi sammen i sanntid. Det virker på samme måte som f.eks. Google Docs og vi deler samme dokumentet. Forskjellen er at vi bruker Microsoft Visual Studio Code. På den måten kan jeg gi deg tilbakemelding umiddelbart og løse kodingen sammen.
